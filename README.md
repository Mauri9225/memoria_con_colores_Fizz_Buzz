“Memoria con colores y FizzBuzz”
Instrucciones:
1. Crea una página que tenga:
    - Un input para el tamaño n de una matriz n x n.
    - Dos inputs numéricos para Fizz y Buzz.
    - Un botón “Generar matriz”.
2. Al hacer click en el botón:
    - Genera una matriz n x n de divs numerados de 1 a n².
    - Si el número es múltiplo de Fizz, la celda tiene un color especial (por ejemplo verde).
    - Si es múltiplo de Buzz, otro color (por ejemplo azul).
    - Si es múltiplo de ambos, otro (por ejemplo morado).
3. Aplica estilos con CSS:
    - Usa display: flex o display: grid para formar la matriz.
    - Agrega transition y :hover para que las celdas cambien suavemente de color o se muevan un poco al pasar el mouse.
4. Muestra encima un texto explicando qué significa cada color.
